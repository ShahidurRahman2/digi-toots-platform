# Digital Tools Buying Website

## 📌 Project Description
A modern and responsive digital tools marketplace where users can browse products, view features, and add items to a cart. The project focuses on clean UI, smooth user experience, and interactive cart functionality.

##  Technologies Used
- React.js
- Tailwind CSS
- DaisyUI
- JavaScript (ES6+)
- React-Toastify
- JSON (Product Data)

##  Features
-  Add to cart & remove product functionality
-  Toast notifications for user actions (add/remove/checkout)
-  Fully responsive design for all devices

##  Additional Features
- Cart count updates dynamically in Navbar
- "Proceed to Checkout" clears all cart items
- Product toggle between Cart & Product sections
